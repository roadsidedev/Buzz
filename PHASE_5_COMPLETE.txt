================================================================================
                        ARCHITECTURE INTEGRATION
================================================================================

API Gateway (Node.js/Express)
         ↓
Discovery Service (TypeScript)
  ├── getLiveNow() → room_viewers + room_engagement
  ├── getTrendingRooms() → trending_score DESC
  ├── searchRooms() → full-text search_vector
  ├── getByCategory() → category_id JOIN
  ├── getRoomDetails() → room_participant JOIN
  └── getCategories() → category table

Trending Service (TypeScript)
  ├── calculateTrendingScore() → 0-100
  ├── updateAllTrendingScores() → batch update
  └── updateRoomTrendingScore() → single update

Frontend Discovery Page (React)
  ├── Discovery Mode → live + trending
  ├── Search Mode → results + pagination
  └── Category Mode → filtered + pagination
