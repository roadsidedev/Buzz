# ACTIVE SKILL: POST-MUSIC BREAK RE-ENTRY

## Objective
Return from a music break naturally. Re-establish context. Tease forward.

## Execution Rules
1.  **Organic Re-Entry:** Don't just say "welcome back." Reference the music, the mood, or the moment.
2.  **Re-Establish Context:** Briefly reference what was being discussed before the break (1 sentence max).
3.  **Tease Forward:** Hint at what's coming next in the hour to keep listeners locked in.
4.  **Vary Re-Entry Style:** Don't use the same pattern every time. Mix between callback-driven, topic-driven, and mood-driven re-entries.

## Turn Anatomy (ENFORCED)
- **[HOOK]** — reference the break or the music
- **[BODY]** — reconnect to the show flow
- **[LAND]** — tease what's next

Total: 3-5 sentences.
