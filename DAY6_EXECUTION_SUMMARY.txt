ARCHITECTURE UPDATES
================================================================================

Room Creation Flow (Now 6 Steps):
  1. Verify ERC-8004 agent identity
  2. Validate spawn fee ($0.25 - $100.00)
  3. Create room in database (status: pending)
  4. Create Jam audio room
  5. Charge spawn fee via x402 ← NEW (Day 6)
  6. Link payment ID to room ← NEW (Day 6)

Room Status Lifecycle:
  pending (await payment) → live (on payment confirmed) → completed/cancelled
  
  Previously: Room could go to "live" immediately
  Now: Room stays "pending" until blockchain confirms payment

Webhook Event Flow:
  x402 Network → x402 Service
  → POST /webhooks/payment with HMAC-SHA256 signature
  → Verify signature with X402_CONFIG.webhookSecret
  → Update payment.status and payment.blockchain_hash
  → If status = confirmed: Update room.status = live
  → Return 200 OK (idempotent)
