## Design System At a Glance

### Color Palette
```
Neutral:    #000000 (black) + #ffffff (white)
Primary:    #0EA5E9 (electric cyan) — interactive, focus, links
Secondary:  #EC4899 (electric magenta) — featured, special
Tertiary:   #F59E0B (electric yellow) — warnings, alerts
Status:     #10B981 (success), #EF4444 (error), #3B82F6 (info)
```

### Typography Stack
```
Display:    Space Grotesk 700 (bold, uppercase headings)
Body:       Inter 400–700 (regular text, 16px)
Mono:       JetBrains Mono (code, technical)
Sizes:      60px (H1) → 14px (small) with aggressive hierarchy
```

### Component Variants
```
Button:     primary (black) | secondary (white) | accent (cyan)
Card:       default | bordered | flat
Badge:      default | primary | secondary | success | warning | error
Input:      text, email, password, number, search (all types)
Textarea:   multi-line with optional char count
```

### Responsive Breakpoints
```
Mobile:     375px+ (default)
Tablet:     768px+ (md:)
Desktop:    1024px+ (lg:)
```

## Design System Philosophy

### Neobrutalism
Raw. Bold. Honest. Unapologetic.

- **Not** minimal design (minimalism is sparse)
- **But** intentional, structural honesty
- **Not** skeuomorphism (realistic, textured)
- **But** digital abstraction with character
- **Not** flat design (lifeless)
- **But** flat aesthetics with strong edges

### Our Implementation
```
Pure black (#000) + Pure white (#fff)
Electric accents (#0EA5E9, #EC4899, #F59E0B)
2px bold borders (not shadows)
Chunky Space Grotesk headings
Uppercase labels with letter-spacing
High contrast, industrial aesthetic
Snappy interactions (< 0.5s)
```
