   - Returns JWT pair (access + refresh tokens)
   - Secure error messages

✅ User Login
   - Email + password authentication
   - Password verification (time-constant)
   - Access token (1 hour)
   - Refresh token (7 days)

✅ Token Refresh
   - Automatic refresh before expiry
   - Scheduled 2 minutes before token expires
   - Single refresh lock (prevents thundering herd)
   - Graceful error handling

✅ Session Management
   - Tokens persisted to localStorage
   - Session recovered on app reload
   - Token validation on app initialization
   - Automatic logout on expiry

✅ Protected Routes
   - ProtectedRoute component (requires authentication)
   - PublicRoute component (blocks authenticated users)
   - RoleRoute component (role-based access)
   - Proper redirects based on auth state

✅ 401 Error Handling
   - Automatic detection on expired token
   - Refresh with retry of original request
   - Prevents infinite redirect loops
   - Graceful logout on refresh failure

✅ Security Hardening
   - Input sanitization (removes dangerous characters)
   - XSS prevention (React escaping + CSP recommendations)
   - CORS configuration (origin whitelist)
   - Password strength validation
   - No token logging or exposure
